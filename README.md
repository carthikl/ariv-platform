## Six Components

### [Overview](https://ariv-platform.vercel.app)
Quality@Speed operating model — five pillars, financial impact framework, AI credentials, and reference repositories.

### [Industry Domain Deep Dives](https://ariv-platform.vercel.app)
Seven technology disciplines across three industries with pluggable cloud (AWS / Azure / GCP) and LLM (Claude / Copilot / OpenAI / Gemini) layers.

**Disciplines:** Application Engineering · Data & Analytics · Infrastructure & Cloud · Platform & Security · SRE & AIOps · Quality Engineering · Governance & Compliance

**Industries:** Discover Financial Services (anchor — full evidence) · Retail Pharmacy · Wealth Management

### [Architecture Explorer](https://ariv-platform.vercel.app)
Seven omnichannel channels × seven stack layers — every combination documented with design patterns, technology decisions, and the architectural trade-off most teams get wrong.

**Channels:** Web · Mobile · Contact Center · Chat/AI · In-Store · Agent Tools · Agentic Commerce

**Layers:** UI · API Gateway · AI/RAG/MCP · Microservices · Data & Events · Infrastructure · Governance

### [Leadership Operating System](https://ariv-platform.vercel.app)
Quality@Speed as a complete executive operating system — not a methodology, a practice.

- **Philosophy** — four leadership pillars, four operating principles
- **Operating Cadence** — Annual → Quarterly → Monthly → Weekly → Daily rhythms
- **Governance** — ARB, TRC, CAB, ESC + Aligned Autonomy framework
- **Team Design** — COE → COP → Federated Embedded Model evolution
- **Measurement** — four metric tiers, financial impact formula, reporting by audience
- **Entry Playbook** — 90-day framework + 30-day architecture dashboard + anti-patterns

### [Maturity Assessment](https://ariv-platform.vercel.app)
18-question scored assessment across six dimensions. Produces a radar chart maturity profile with prioritized recommendations linked to relevant ARIV components.

**Dimensions:** Delivery Velocity · Quality Engineering · AI Readiness · Governance & Compliance · Organizational Health · Technology Platform

### [Intelligence Layer](https://ariv-platform.vercel.app)
RAG, MCP, A2A, and UCP — the extensible intelligence connectors that make enterprise AI deployable, governable, and auditable.

- **RAG** — Retrieval Augmented Generation: ground AI in enterprise knowledge
- **MCP** — Model Context Protocol: give Claude structured access to enterprise systems
- **A2A** — Agent-to-Agent: multiple agents coordinating across organizational boundaries
- **UCP** — Universal Commerce Protocol: cross-platform transaction execution

All four layers governed by the AIGP AI Governance Professional framework.
